# Bank_Churn_Analysis
📊 Bank Customer Churn Analysis

📌 Project Overview
This project analyzes customer data from a bank to identify patterns and factors that lead to customer churn (customers leaving the bank).
The goal is to help businesses reduce churn and improve customer retention using data-driven insights.

🎯 Objectives
Identify key factors affecting churn
Analyze customer demographics and behavior
Visualize churn trends
Build insights for business decision-making
🛠️ Tools & Technologies
Python (Pandas, NumPy)
Power BI (Dashboard)
Excel (Data preprocessing)

📂 Dataset
Contains customer details like:
Age
Balance
Tenure
Credit Score
Geography
Churn Status

📈 Key Insights
Customers with low balance are more likely to churn
Churn rate is higher in certain regions
Older customers show higher churn tendency
Active members churn less compared to inactive users

🚀 Project Workflow
Data Cleaning
Exploratory Data Analysis (EDA)
Visualization
Dashboard Creation (Power BI)

📌 Conclusion
The analysis helps banks identify at-risk customers and take preventive actions such as personalized offers, improved services, and engagement strategies.
